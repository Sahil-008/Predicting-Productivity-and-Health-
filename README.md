# Productivity & Health Prediction Project

This project predicts whether a person is **productive and healthy** based on social media usage
and lifestyle-related features.

## 📂 Project Structure

```
notebooks/
├── 01_Data_Overview_and_Cleaning.ipynb
├── 02_EDA_and_Visualization.ipynb
├── 03_Feature_Engineering_and_Label_Creation.ipynb
├── 04_Model_Training_LogisticRegression.ipynb
├── 04_Model_Training_RandomForest.ipynb
├── 04_Model_Training_SVM.ipynb
├── 04_Model_Training_XGBoost.ipynb
└── 05_Model_Comparison.ipynb

data/
├── raw/
├── processed/

results/
└── model_metrics.csv
```

## 📌 Workflow

1. **Data Cleaning & Outlier Handling**
2. **EDA & Visualization**
3. **Feature Engineering + Label Creation**
4. **Model Training**
5. **Model Comparison**
6. **Final Results + Reporting**

## 🧪 Models Used
- Logistic Regression 
- Linear Regression 
- Random Forest  
- LightGBM  
- XGBoost  

Metrics include:
- Accuracy  
- Precision  
- Recall  
- F1-score  
- ROC-AUC  

## 📁 Results
All evaluation metrics are stored in:
```
results/model_metrics.csv
```

## ✨ Goal
“The goal of this project is to develop a machine learning–based predictive framework that analyzes behavioral factors such as screen time, workload, and lifestyle patterns to estimate a user’s productivity and health state, and to validate its effectiveness in supporting well-being and performance management.”
